# Webcompat dashboard editor testing

Webcompat dashboard editor testing

### Quick Setup

```console
git clone git@github.com:webcompat/webcompat-editor-testing.git
cd webcompat-editor-testing
npm install
```

## Development

```console
npm run dev
```

## Static build

```console
npm run build
```
